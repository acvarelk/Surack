# Surack
Osint-инструмент для поиска информации по IP, номеру телефона, никнейму.

- Install (For Windows)
    - ```git clone https://github.com/acvarelk/Surack```
    - ```cd Surack```
    - ```pip install -r requirements.txt```
    - ```cd win```
    - ```python main.py```
